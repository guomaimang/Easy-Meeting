# 设置中心

## 目标

设置中心用于集中管理 Easy Meeting 的常用配置，菜单栏仅保留高频快捷入口。

## 信息架构

设置窗口采用左侧栏目、右侧配置区：

- 程序：管理悬浮字幕窗透明度、字幕字体大小等应用本体行为。
- 语音：管理语音服务商、翻译模式、源语种、目标语种和火山引擎 API Key，并展示固定 Resource ID。
- 麦克风：管理麦克风权限、刷新输入设备、选择当前输入源。

## 行为约定

- 菜单栏仅保留麦克风、翻译模式等快捷操作；悬浮窗透明度改由设置中心滑块和 `Command + +` / `Command + -` 快捷键调整，不再放入菜单栏下拉。
- 完整配置必须能在设置中心集中查看和调整。
- 设置实时自动保存，不提供独立的"保存"按钮：
  - 下拉框（语音服务、翻译预设、源语种、目标语种、输入设备）选择即保存。
  - 文本框（API Key、区域 Region）在失去焦点或回车时保存，避免逐字符写 Keychain。
  - 滑块（透明度、字体大小）拖动时实时预览，松手时保存。
  - 粘贴、清空 API Key 等按钮操作立即保存。
- 任意改动持久化成功后，底部状态行短暂显示"已保存"，失败显示错误信息。
- 程序化加载设置（打开窗口、切换服务商重填下拉框）不触发自动保存。
- 透明度保存到 `UserDefaults`，下次启动恢复。
- 字幕字体大小保存到 `UserDefaults`，调整后立即作用于悬浮字幕窗。
- 悬浮字幕窗接收鼠标滚轮和拖拽事件，用于翻阅字幕、移动位置和调整尺寸。
- 悬浮字幕窗使用左右两栏展示原文和译文，两栏分别维护滚动位置，避免中英文长度不一致时相互牵动。
- 悬浮字幕窗必须设置为不可共享窗口，降低全屏共享、录屏和截图时被采集的概率。
- 悬浮字幕窗支持鼠标拖动位置、边缘拖拽缩放，也保留全局快捷键调整位置和透明度：
  - `Command + 方向键`：移动悬浮窗。
  - `Command + +` / `Command + =`：提高悬浮窗不透明度（更清晰），每次 0.05。
  - `Command + -`：降低悬浮窗不透明度（更透明），每次 0.05。
  - `Command + 0`：恢复默认位置和尺寸，不修改字幕字体大小。
  - 不透明度取值钳制在 0.1–1，调整后立即生效并写入 `UserDefaults`。窗口缩放仅保留鼠标边缘拖拽，不再提供键盘快捷键。
- 火山引擎 API Key 保存到 Keychain。
- 火山引擎 AST 2.0 的 Resource ID 固定为 `volc.service_type.10053`，设置页只展示，不要求用户填写。
- Azure 语音密钥保存到 Keychain，Region 默认预设为 `eastasia`；Region 为空时也必须按默认值启动，不允许阻断录音。
- AST S2T 语言配置保存到 `UserDefaults`，录音启动时按设置页保存的翻译模式传递给 helper。
- 粤语按官方参数传 `yue-CN`，中英反转互译按官方参数 source/target 都传 `zhen`。
- Azure 翻译预设仅保留英文转中文、中文转英文、粤语转中文三项；粤语按 Azure 源语种参数传 `yue-CN`。
- 源语种下拉展示官方 S2T 支持的 20 个外语、2 个方言和 `zhen`。
- 目标语种下拉展示官方 S2T 支持的 20 个外语和 `zhen`，不展示只能作为源语种的方言。
- 设置窗口内文本输入必须支持系统标准编辑命令，包括复制、粘贴和全选。
- 点击设置页空白区域时，当前文本输入框必须结束编辑并失去焦点。
- API Key 输入必须适配长密钥，提供足够宽的单行输入区、粘贴、显示/隐藏、清空和长度反馈。
- 设置页采用类似系统偏好设置的左侧栏目、右侧分组行布局，避免临时表单感。
- 左侧栏目使用灰色侧栏、分组标题、图标和蓝色选中态，保持 macOS 偏好设置风格。
- 左侧栏目每个入口必须使用系统按钮事件链、固定行高，整行点击都能切换页面，不允许被 `NSStackView` 拉伸成大块选区。
- 设置窗口右侧内容必须控制在紧凑分组内，控件和值标签对齐，不允许出现大片失衡空白或文字遮挡图标。
- 麦克风设备选择作用于当前运行会话，设备列表由系统音频设备发现结果刷新。

## 验证标准

- 打开设置窗口时，左侧能切换“程序 / 语音 / 麦克风”。
- 左侧栏目具备清晰的分组、图标和选中高亮，不使用裸按钮堆叠，点击图标、文字或行内空白都能切换。
- 左侧选中态高度稳定为单行，不随窗口高度或内容页变化而拉伸。
- 设置页控件排列紧凑，分组背景、分隔线、标题和值标签在浅色窗口中可辨认。
- 调整透明度后，悬浮字幕窗立即更新，松手自动保存后重启可恢复。
- 调整字幕字体大小后，悬浮字幕窗立即更新，松手自动保存后重启可恢复。
- 使用鼠标或全局快捷键可以移动、缩放、重置悬浮字幕窗。
- 悬浮字幕窗能同时展示原文和译文两栏，窗口尺寸变化时两栏保持可读，左右滚动互不影响。
- 使用系统截图、录屏或常规会议共享能力采集屏幕时，悬浮字幕窗不应作为可共享窗口暴露。
- 修改语音服务和 API Key 后自动保存，可通过"检查配置"查看 helper 状态。
- 修改翻译模式后，设置页和菜单栏入口使用同一份保存配置。
- 设置页可以直接选择源语种和目标语种，录音启动时实际使用这两个语种参数。
- 语种组合错误提示必须在独立提示行展示，不允许在行内溢出或遮挡控件；语种组合非法时不写入持久化。
- API Key 输入框可以通过 `Command-V` 粘贴外部复制的密钥。
- API Key 输入框可以通过按钮从剪贴板粘贴、切换显示状态、清空内容，并显示当前长度。
- 在设置页空白处点击后，API Key 输入框不再保持焦点，并触发失焦自动保存。
- 麦克风页可以请求权限、刷新设备、选择输入设备。
